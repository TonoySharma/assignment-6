1= Project Name :DigiTools-Platform

2= Description : 👉 This is a digital tools purchasing website where users can easily explore different cards/tools. They can add their preferred cards to the cart and complete purchases very easily. The website is fully responsive, meaning it looks great and works smoothly on mobile, tablet, and laptop devices.This website is built using React, Tailwind CSS, JavaScript (ES6), React Icons Toastify, and DaisyUI. As a result, the website looks very attractive and provides a 
smooth user experience.

3= Technologies Used :
- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6)
- React,icon, Toastify

4= 1. Users can add products to cart and see total count in navbar  
2. Users can remove individual products or clear cart using checkout  
3. Toast notifications show when adding, removing, or checking out products
